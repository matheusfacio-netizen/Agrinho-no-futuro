# Agrinho-no-futuro
O projeto "Inteligência Artificial no Campo" é um site educativo que mostra como drones, GPS e inteligência artificial estão modernizando a agricultura. Além de informações sobre essas tecnologias, o site conta com um jogo simples em que um drone coleta alimentos, tornando o aprendizado mais interativo e divertido. 🚜🤖🌱
